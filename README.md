## 使用技术

前端html，后端使用php接口传递数据，数据库使用leancloud。

使用的插件：

| 名称       | 作用               |
| ---------- | ------------------ |
| wangeditor | 简洁的富文本编辑器 |
| axios      | 数据请求           |
| qs         | 数据序列化         |
| smoke      | 美化的消息提醒     |
